# About
Asteroids game for practice. Pretty simple game.  

## Controls  
### Movement  
W = (move ship up)
A = (move ship down)  
S = (move ship left)  
D = (move ship right)  
SPACEBAR = (shoot)  

## Game Over
Game will end once ship collides with a space rock, spcace cowboy.  
Currently, there is no "winning" the game, just exit once bored or you die.
